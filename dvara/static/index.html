dvara

High-speed malicious URL detection using a probabilistic Bloom Filter pipeline. Precision security for technical architectures.

About the Creator

Built by Dhruv, an engineering student at BMSCE Bangalore with a focus on systems design and ML. dvara started as a curiosity about how probabilistic data structures could solve a real security problem, turns out, they can. It's a working tool, built from scratch, deployed, and installable right now with one command.
